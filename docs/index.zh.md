# MetaD-toy: 面向 LAMMPS 的高性能元动力学插件

欢迎查阅 **MetaD-toy** 官方技术文档！这是一个专为 LAMMPS 分子动力学框架设计的非侵入式加速扩展插件。

本插件融合了底层的 C++/CUDA 异构并发架构，专门用于加速多组元金属体系（如铝基体、钨合金）在**凝固形核、多相转变、魔数团簇演化及晶型选择**过程中的自由能势垒跨越与非平衡态采样。

---

## 🚀 核心技术特性

* **GPU 并发加速 (CUDA)**：针对局域键级取向序参数（如 $Q_4, Q_6$）以及复杂的化学短程序因子，提供原子级的高并发显存直连求解。
* **智能网格拓扑**：内置“均匀密集网格”与**“去饱和稀疏哈希自由能网格”**双引擎，彻底杜绝高维、大生命周期模拟时的显存爆炸（VRAM Overflow）。
* **去中心化数据总线**：配套开发的 `compute metad/atom` 节点采用多态动态路由设计，支持以 $O(N)$ 的高吞吐量无缝导出原子级瞬时序参数。
* **双语规范合规**：代码白盒结构完备，文档严格对齐，全面支持科研同行评议与国家计算机软件著作权申报。

## 🛠️ 快速挂载示例

在你的 LAMMPS 输入脚本中，只需通过 `plugin load` 载入动态链接库，即可无缝激活元动力学偏置：

```lammps
# 1. 载入动态链接库
plugin load ${METAD_PLUGIN_PATH}/fix_crystallize_plugin.so

# 2. 声明并启动 2 维元动力学系统
fix m all metad GAUSSIAN 0.003 0.05 10.0 PACE 1 CV_dim 2 ...
```
